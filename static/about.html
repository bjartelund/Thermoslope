A webapp to take UV-VIS absorbance reading done over a time/temperature gradient to estimate temperature-dependent velocities, giving a single experiment estimate of the Arrhenius parameters and thus the thermodynamic parameters of the activation barrier.

Go to the submission-page to upload your data files and experimental parameters. A random ID will be generated for your session and data files. The analysis can for some time be retrieved with this ID, but it is not persistant and no backups are performed.

For now only data-files for the Agilent Cary 60 with the QNW qChanger6 with modified software is supported. But it should be possible to support other units where the temperature can be ramped.

Follow the development at GitHub. At GitHub you can also find example data for input.

Publication/preprint will come soon(ish).
